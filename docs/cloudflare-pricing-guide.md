# Cloudflare Pricing Guide

This project does not require an exhaustive pricing document to get started.

## Practical Recommendation

- start with the smallest plan that covers your real traffic
- keep optional services disabled until needed
- watch D1, Images, AI, and Vectorize usage first
- avoid premature enterprise planning

## For This Boilerplate

The most important cost drivers are usually:

- Worker requests
- D1 usage
- Images storage and transformations
- AI or Vectorize calls
- optional R2 storage

## Recommendation

Use Cloudflare’s current official pricing pages when you are making a real purchasing decision. Pricing changes over time and should not be hardcoded into repository docs.
